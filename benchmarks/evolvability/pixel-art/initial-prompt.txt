Create a pixel-art program in React. Users can select color from an RGB color-picker, and then click on the drawing area to paint the color. They may also save the current drawing as a bitmap on the local file system, and load bitmaps from the local file system.

Configure it so that running `npm run start -- --port=$PORT` opens the app in a browser window with port $PORT.
Pay attention to code quality, as you will be evaluated on how many lines must be changed to add new features.
Do not implement any more features than you have been asked.
